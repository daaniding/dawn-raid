# dawn raid - Claude instructies

## Stack
- Next.js App Router, TypeScript
- Tailwind CSS v4
- Upstash Redis
- PixiJS voor game rendering
- Vercel voor deployment

## Regels
- Push altijd naar main
- Deploy via vercel --prod
- Test altijd of dev server start na wijzigingen
- Scan /public/assets/ voor je sprites gebruikt
- Verzin nooit bestandsnamen die niet bestaan
- Na elke taak: benoem welke bestanden zijn aangepast

## Assets
Staan in /public/assets/ met submappen per pack.
Scan deze map altijd eerst voor je begint.

## Wat we bouwen
Lees GAME.md voor het volledige concept.
We bouwen stap voor stap, week voor week.
